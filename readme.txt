Preenchimento automático de campos de endereço baseado no CEP informado.

== Description ==

Este é um simples e leve plugin para preencher o endereço pelo CEP de forma automática.
É voltado para o plugin woocommerce.

**Woocommerce Preenche endereço CEP** identifica o campo que irá receber o CEP e os campos a serem preenchidos automaticamente através dos nomes de elementos via jquery que informam ao plugin o valor que o campo deve receber.

Não há nenhuma necessidade de configuração.


= Dúvidas? =

Você pode esclarecer suas dúvidas criando um tópico no [fórum de ajuda do WordPress](https://wordpress.org/support/plugin/woocommerce-preenche-endereco-cep).

= Créditos =

Foram utilizados os seguintes scripts/serviços de terceiros:

* [ViaCEP](https://viacep.com.br/).

== Installation ==

1. Faça o upload dos arquivos do plugin no diretório `/wp-content/plugins/woocommerce-preenche-endereco-cep`, ou instale diretamente pela área de plugins do WordPress.
2. Ative o plugin.
